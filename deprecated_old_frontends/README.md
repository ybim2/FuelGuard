Deprecated frontend archive
===========================

This folder contains frontend files that are not part of the canonical Fuel Guard mobile PWA.

Do not import, edit, or revive files from this folder unless the user explicitly asks to recover an archived implementation.

The canonical app is the root-level mobile-first PWA rendered by `index.html` with the three main tabs:

- Rhythm
- History
- Settings

Archived files should stay out of the active app shell and service worker cache.
